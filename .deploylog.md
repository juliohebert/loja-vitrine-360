# 🚀 Deploy - 27 de Abril de 2026

## 📋 Alterações Incluídas Neste Deploy

### ✨ Novas Funcionalidades

1. **Onboarding Completo**
   - Fluxo de boas-vindas para novos usuários
   - Configuração inicial da loja
   - Upload de logo durante cadastro
   - Definição de estilo do PDV

2. **Catálogo Público com Melhorias**
   - Carrossel de imagens para produtos com múltiplas fotos
   - Navegação por setas (← →)
   - Indicadores de página (dots)
   - Contador de imagens (1/3, 2/3, etc)
   - Sistema de slugs para URLs amigáveis

3. **Sistema de Produtos Aprimorado**
   - Modal de sucesso com opções pós-cadastro
   - "Cadastrar Novo Produto" ou "Ver Produtos"
   - Campo Marca agora é opcional
   - Campo Preço de Custo agora é opcional
   - Correção de inputs numéricos (remove zeros à esquerda)

4. **Gestão de Opções de Produtos**
   - Cadastro rápido de categorias inline
   - Cadastro rápido de subcategorias inline
   - Cadastro rápido de marcas inline
   - Rotas de API implementadas

### 🐛 Correções de Bugs

1. **Configurações de Loja**
   - Corrigido bug onde nome da loja não salvava após onboarding
   - Corrigido async/await em `saveOnboardingConfig`
   - Logo agora persiste corretamente

2. **Catálogo Público**
   - Corrigido campo `slug_catalogo` que estava NULL
   - Script `fix-slug-catalogo.js` para corrigir dados existentes
   - Rotas 404 corrigidas para `/catalogo/:slug/*`

3. **Validação de Produtos**
   - Corrigido erro "Campos obrigatórios" mesmo com campos preenchidos
   - Backend e frontend agora aceitam marca e precoCusto opcionais
   - Valores padrão: marca="Sem marca", precoCusto=0

4. **Inputs Numéricos**
   - Removido comportamento de zero à esquerda (05 → 5)
   - Mudado de type="number" para type="text" + pattern
   - Validação com regex para aceitar apenas números

### 🔧 Melhorias Técnicas

1. **Logs de Debug**
   - Logs detalhados em `productController.js`
   - Logs detalhados em `catalogoController.js`
   - Rastreamento de `exibir_catalogo` em create/update

2. **Rotas de API**
   - `/api/products/options` para buscar opções
   - `/api/products/options/:type` para adicionar opções
   - Ordem de rotas corrigida (específicas antes de `:id`)

3. **Scripts Utilitários**
   - `backend/fix-slug-catalogo.js` - corrige slugs no banco
